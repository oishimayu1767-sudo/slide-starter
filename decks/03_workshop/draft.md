# BIワークショップ企画案（第1回：事例共有＆課題棚卸し編）

## 1. 開催趣旨

先進企業の事例から「BIツールで何ができるのか」「導入時にどんな壁があるのか」を具体的に学びます。他社との意見交換を通じて、自社導入に向けた「最初の一歩（どのデータを、何のために可視化するか）」を明確にすることを目的とします。

## 2. 開催概要

- ターゲット： 高知DXコミュニティ参加企業（経営層・現場担当者）
- 開催時間： 120分
- 実施形式： 対面（会場：トミナガ会議室）
- 定員： 10名程度（5名×2グループを想定）

## 3. プログラム構成（120分）

| 時間 | 項目 | 内容・ポイント |
|------|------|--------------|
| 5分 | 開会挨拶 | 趣旨説明、本日のゴール設定 |
| 15分 | 事例共有①（垣内様） | 現場での活用事例（ダミーデータ）。 |
| 10分 | 質疑応答 | 参加者からの技術的・組織的な疑問の解消 |
| 15分 | 事例共有②（土佐電子様） | 現場での活用事例（ダミーデータ）。 |
| 10分 | 質疑応答 | 参加者からの技術的・組織的な疑問の解消 |
| 5分 | 休憩 | ネットワーキング、グループ分けの調整 |
| 35分 | 【ワーク】意見交換 | 各社課題を共有、グループで話し合いワークシートにまとめる。最終的にどんなワークショップがいいかまとめる |
| 20分 | 全体共有・フィードバック | 各グループの気づきをシェアし、共通の悩みを可視化。今後のワークショップ案を提案 |
| 5分 | 閉会挨拶 | アンケート実施、第2回に向けたアナウンス |

## 4. 事前準備（アクションアイテム）

精度の高いワークショップにするため、参加者へ以下の事前アンケートを実施します。

- データの蓄積状況： 紙、Excel、クラウドシステム、基幹システム等、どの段階か？
- 具体的に知りたい事例： 製造工程の効率化、売上予測、原価管理、在庫適正化など。
- 現場の悩み： 「同じデータを二重入力している」「集計に月10時間かかっている」等の具体例。
- 利用環境： すでに導入検討しているツールがあるか（Power BI, Tableau, Looker Studio等）。

## 5. 今後の展開

今回のワークショップで出た「可視化したいデータ」や「共通の課題」を元に、第2回では実践型の「ステップアップ・ワークショップ」へと繋げます。
